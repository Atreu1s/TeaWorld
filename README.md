# TeaWorld 

Образовательный веб-проект о чае с системой управления контентом (CRUD)

## Описание

**TeaWorld** — это учебный проект, посвящённый истории и разнообразию чая. Сайт позволяет изучать основные виды чая, читать интересные статьи в блоге и управлять контентом через удобную админ-панель (CRUD).

## Основные функции

- Изучение истории чая и его видов
- Блог с интересными статьями о чае
- Система CRUD для управления контентом
- Система авторизации пользователей
- Современный и адаптивный дизайн

## Технологии

- **Frontend:** React.js
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Styling:** Scss
- **Authentication:** JWT
